Connect to Supabase

Your activity log will sync to the cloud so every device in the office sees the same data. Setup takes about 3 minutes and is completely free.

1
Go to supabase.com → click Start your project → sign up for free (no credit card needed).
2
Create a new project. Give it any name (e.g. office-log) and set a database password. Wait ~1 minute for it to provision.
3
In your project dashboard, click SQL Editor in the left sidebar, paste the query below, and click Run:
create table activities (
  id bigint generated always as identity primary key,
  title text not null,
  category text,
  start_time timestamptz,
  end_time timestamptz,
  notes text,
  logged_at timestamptz default now()
);
4
Go to Project Settings → API. Copy your Project URL and anon public key and paste them below.
