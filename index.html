الاتصال بـ Supabase

سيتم حفظ سجل النشاطات في السحابة حتى تتشارك جميع أجهزة المكتب نفس البيانات. الإعداد يستغرق حوالي 3 دقائق وهو مجاني تمامًا.

1
اذهب إلى supabase.com ← اضغط Start your project ← سجّل حسابًا مجانيًا (لا يلزم بطاقة ائتمان).
2
أنشئ مشروعًا جديدًا، أعطه أي اسم مثل office-log، وضع كلمة مرور للقاعدة. انتظر دقيقة حتى يكتمل الإعداد.
3
في لوحة التحكم اضغط SQL Editor من القائمة الجانبية، الصق الكود أدناه واضغط Run:
create table activities (
  id bigint generated always as identity primary key,
  title text not null,
  category text,
  start_time timestamptz,
  end_time timestamptz,
  notes text,
  logged_at timestamptz default now()
);
4
اذهب إلى Project Settings → API، انسخ Project URL وanon public key والصقهما أدناه.
